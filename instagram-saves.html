Bookmarklet setup

Drag the button below to your bookmarks bar. While on any Instagram post, click it and the app will open with the caption and creator pre-filled.

Chrome install (drag doesn't always work):
1. Right-click the button above → Copy link address
2. Open Chrome bookmarks bar → right-click → Add page...
3. Name it "→ Save to Notion", paste the copied link as the URL → Save

How to use: open any Instagram post, click the bookmark in your bar — this app opens in a new tab with caption and creator pre-filled.
